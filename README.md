# Coupled Double-Bracket Gradient Flows on Adjoint Orbits (AOAD)

Reference implementation accompanying the paper:

> Jerina Jeneth C. Francisco  
> **Coupled Double-Bracket Gradient Flows on Adjoint Orbits: Commutator Energy
> Dissipation, Algebraic Confinement, and Asymptotic Alignment** (2026)  
> Submitted to *SIAM Journal on Matrix Analysis and Applications*

---

## Overview

This repository contains the simulation code used to illustrate and investigate
the numerical behavior of the coupled isospectral gradient system studied in
the AOAD paper.

The implementation is intended to support the theoretical analysis presented
in the manuscript and should not be interpreted as an independent proof of
the mathematical results.

---

## Implemented Dynamics

All simulations in this repository implement the coupled AOAD system exactly
as stated in the manuscript:

```
L_G' = -α [L_G, [L_G, L_J]]
L_J' = -β [L_J, [L_J, L_G]]
```

with α, β > 0, on products of adjoint orbits of real symmetric n×n matrices.

The core implementation is in `aoad/dynamics.py`. The numerical experiments
are intended to provide reproducibility support for the theoretical analysis
and should not be interpreted as independent proofs of the mathematical results.

---

## Reproducibility Artifacts

### Section 9: Explicit 2×2 Example

The primary reproducibility artifact is the explicit 2×2 system analyzed in
Section 9 of the paper. For initial conditions

```
L_G = diag(1, -1),    L_J = [[0, 1], [1, 0]]
```

with α = β = 1, the linearized theory (Theorem 8.2) predicts asymptotic
energy decay with rate 2γ_min = 8, where

```
γ_min = α(λ₁ − λ₂)² + β(μ₁ − μ₂)² = 4
```

The supplied code reproduces the predicted exponential decay to numerical
precision and verifies the convergence rate γ_min = 4 predicted by the
linearized theory (Theorem 8.2).

### General n×n Simulations

The implementation also supports arbitrary real symmetric initial data
and is intended to illustrate:

- Spectral preservation along the flow (isospectrality)
- Evolution of the commutator energy `E = ½ ‖[L_G, L_J]‖_F²`
- Approach toward commuting configurations
- Dependence on anisotropic weights α, β
- Confinement inside the Lie algebra generated by initial conditions

## Running the Code

Install dependencies:

```bash
pip install numpy scipy matplotlib
```

Reproduce the Section 9 explicit example:

```bash
python examples/example_2x2.py
```

Run a random symmetric matrix experiment:

```bash
python examples/example_random.py
```

Run the anisotropic weighting comparison:

```bash
python examples/example_anisotropic.py
```

Reproduce the local rate verification (Theorem 8.2):

```bash
python examples/plot2_local_rates.py
```

## Companion Paper

This repository also contains `double_bracket_flow.py`, the simulation
code for the companion work:

> Jerina Jeneth C. Francisco  
> **Invariant Lie Algebra Confinement in a Symmetric Double-Bracket Flow** (2026)  
> DOI: [10.5281/zenodo.20325811](https://zenodo.org/records/20325811)

The companion paper studies the confinement phenomenon for the unweighted
system (α = β = 1). The AOAD paper extends this framework to anisotropic
coupled flows on products of adjoint orbits and establishes global convergence
and exponential asymptotic alignment.
