# Coupled Double-Bracket Gradient Flows on Adjoint Orbits (AOAD)

Reference implementation accompanying the paper:

> Jerina Jeneth C. Francisco  
> **Coupled Double-Bracket Gradient Flows on Adjoint Orbits: Commutator Energy
> Dissipation, Algebraic Confinement, and Asymptotic Alignment** (2026)  
> Submitted to *SIAM Journal on Matrix Analysis and Applications*

---

## Overview

This repository contains the simulation code used to reproduce the numerical
and illustrative results presented in the AOAD paper.

The paper studies the coupled isospectral gradient system

```
L_G' = -α [L_G, [L_G, L_J]]
L_J' = -β [L_J, [L_J, L_G]]
```

on products of adjoint orbits of real symmetric matrices. The implementation
demonstrates the principal theoretical results:

- Exact preservation of eigenvalues (isospectrality)
- Monotonic decay of commutator energy `E = ½ ‖[L_G, L_J]‖_F²`
- Confinement to the Lie algebra generated by the initial data
- Convergence toward commuting equilibria
- Exponential local convergence at the rate predicted by the linearized theory

## Reproducibility Artifacts

### Section 8: Explicit 2×2 Example

The main reproducibility artifact is the explicit 2×2 system analyzed in
Section 8 of the paper. For

```
L_G = diag(1, -1),    L_J = [[a, b], [b, -a]]
```

the theory predicts

```
b(t) = b₀ exp(-4t)
E(t) = 4b₀² exp(-8t)
```

The supplied code reproduces this decay exactly and verifies the predicted
convergence rate `γ_min = 4`, confirming sharpness of Theorem 6.2.

### General n×n Simulations

The implementation also supports arbitrary real symmetric initial data and
allows verification of:

- Spectral preservation along the flow
- Commutator energy dissipation
- Convergence toward commuting pairs
- Dependence on anisotropic weights α, β
- Confinement inside the Lie algebra generated by initial conditions

## Running the Code

Install dependencies:

```bash
pip install numpy scipy matplotlib
```

Reproduce the Section 8 example:

```bash
python examples/example_2x2.py
```

Run a random symmetric matrix experiment:

```bash
python examples/example_random.py
```

Run the anisotropic weighting example:

```bash
python examples/example_anisotropic.py
```

## Companion Paper

This repository also contains functionality related to the companion work:

> Jerina Jeneth C. Francisco  
> **Invariant Lie Algebra Confinement in a Symmetric Double-Bracket Flow** (2026)  
> DOI: [10.5281/zenodo.20325811](https://doi.org/10.5281/zenodo.20325811)

The companion paper studies the confinement phenomenon for the unweighted
system (α = β = 1) in greater detail. The AOAD paper extends this framework
to anisotropic coupled flows on products of adjoint orbits and establishes
global convergence and exponential asymptotic alignment.
